# Tactical Compass Overlay

## Developed by: GAMER\_PILLA

### "One deaf to another: happy gaming."



An audio-visual overlay designed to assist hard-of-hearing and deaf gamers by analysing real-time directional audio (stereo balance and frequencies)

and translating it into clean, responsive visual alerts (arcs) on your screen.





##### 🛠️ First-Time Configuration Steps

1\. **Run as Administrator:** Right-click `tactical\\\_arcs\\\_final\\\_release.exe` and select \*\*Run as Administrator\*\*. This grants the application permissions to stay pinned on top of secure game engines.

2\. **Audio Source Selection**: Choose your primary active Headset or desktop audio device from the dropdown loopback menu.

3\. **Monitor Assignment:** Select the target monitor where your game is actively rendered.

4\. **Load Competitive Presets**: Select a quick profile from the dropdown for titles like Call of Duty: Warzone, Fortnite, or Apex Legends to automatically calibrate frequency bands and audio gates, or use the sliders for manual fine-tuning.

5\. **Launch Engine:** Click LAUNCH TACTICAL COMPASS. The main configuration panel will instantly minimize to the system tray, and the visual overlay will go live.



\---



##### ⚠️ CRITICAL GAME SETTINGS REQUIREMENT

Display Mode: Your game MUST be set to Borderless Windowed or Windowed mode.

If you run your game in Exclusive Fullscreen, Windows grants the game full hardware control over the screen graphics, completely blocking the tracking arcs from showing up.

##### 

##### 🎛️ Controlling the App Mid-Game

To Reconfigure / Adjust Sliders: Expand your Windows taskbar system tray icons (near the clock), right-click the Red Ear Icon, and select "Open Configuration Panel" to pop the main menu back up.

To Shut Down Safely: Click "CLOSE VISUALIZER COMPLETELY" in the main menu, or right-click the system tray icon and select "Exit Completely". This completely frees all memory handles, audio hooks, and visual processing threads safely without crashing.

##### 🔒 Anti-Cheat \& Safety Warning

Because this application creates a transparent third-party window overlay to display accessibility graphics, modern anti-cheat systems (such as RICOCHET or Easy Anti-Cheat) may flag it under automated heuristic scans. The software does NOT modify game files, inject code, or read game memory. However, automated bans are difficult to appeal, so users should use the software at their own risk.
